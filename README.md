# SecuraMem IP Acquisition Landing Page

A beautiful, engaging landing page for SecuraMem's IP acquisition campaign, built with Next.js and featuring stunning animations inspired by award-winning designs.

## Features

- **Beautiful Animations**: Framer Motion powered animations with scroll triggers, stagger effects, and micro-interactions
- **Modern Design**: Glass-morphism effects, gradient backgrounds, and particle animations
- **Fully Responsive**: Mobile-first design that looks great on all devices
- **Performance Optimized**: Built with Next.js 15 and React 18 for optimal performance
- **Type Safe**: Full TypeScript implementation

## Tech Stack

- **Framework**: Next.js 15 with App Router
- **Styling**: Tailwind CSS 3.4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## Getting Started

### Installation

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build

Build for production:

```bash
npm run build
```

### Start Production Server

```bash
npm start
```

## Animation Features

- **Scroll-triggered animations**: Content fades and slides in as you scroll
- **Stagger effects**: Children elements animate in sequence
- **Hover interactions**: Cards lift, scale, and show shadows on hover
- **Rotating elements**: Icons rotate on hover
- **Progress indicator**: Top bar shows scroll progress
- **Floating particles**: Background particle animation
- **Gradient animations**: Animated gradient backgrounds
- **Micro-interactions**: Buttons pulse, scale, and glow on interaction

## Customization

The landing page is fully customizable. Key areas to modify:

- **Content**: Edit the data in `app/page.tsx`
- **Colors**: Modify Tailwind config in `tailwind.config.ts`
- **Animations**: Adjust Framer Motion variants in `app/page.tsx`
- **Styles**: Update global styles in `app/globals.css`
